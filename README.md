# Sentenced Differently: Racial and Gender Disparities in Federal Sentencing
This project uses the U.S. Sentencing Commission's publicly available individual offender datasets to test whether race and gender predict sentence length after controlling for legally relevant variables like offense severity, criminal history score, and guideline range. The sentencing guidelines are designed to standardize outcomes, yet published research consistently finds residual disparities. We replicate and update that literature with the most recent data (FY2020-FY2024), run log-linear regression models, and conduct subgroup analysis by offense category to identify where disparities concentrate. Final deliverables include an executed Jupyter notebook, interactive HTML dashboard, multi-page project website, presentation slides, and a written report.

In this project, team members will contribute through roles in literature review, statistical analysis, and subgroup/geographic analysis, while the project manager leads the end-to-end technical pipeline. Final deliverables include a Jupyter notebook, interactive dashboard, research report, and presentation synthesizing both quantitative findings and policy implications.
